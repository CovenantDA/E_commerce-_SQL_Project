#  E-commerce Performance Dashboard



##  Overview
A 2018 retail analysis combining **SQL** and **Power BI** to visualize sales, customer behavior, product performance, and profitability.

##  Key Insights
- **Total Sales:** ₦8M | **Profit:** ₦3.61M | **Quantity Sold:** 128K  
- **Top Products (Shipping Cost):** T-Shirts, Watches  
- **Customer Trends:** Female users lead in purchases  
- **Profit Drivers:** Few high-margin products  
- **Patterns:** Clear monthly/weekly spikes in sales  
- **Delivery Insight:** Priority affects delivery speed

##  Files Included
- `e_commerce-project.sql` – SQL for data analysis  
- `e-commerce project.pbix` – Power BI visuals & KPIs
